Git flow

GIT clone- для того, чтобы"склонить" репозиторий

GIT pull-для того, чтобы получить изминения с репозитория

GIT checkout -b branchname- создать новую ветку и переключитьсяна неё

GIT add .-добавить измения, которые сделали локально

GIT commit -m "commit tex"- подписать их

GIT push origin branchname- запушить на GIT сервер
